# JMDC Synthetic Raw Dataset

This folder holds a **synthetic** JMDC-shaped raw dataset (7 CSVs, 1,000 members)
used to demo and regression-test the JMDC analysis modules (J1~J7, PRD §17).
No real patient data — every row is generated by `scripts/generate_jmdc_synthetic.py`.

## Regenerate

```bash
python scripts/generate_jmdc_synthetic.py
```

Seed is fixed (`SEED = 42`), so output is reproducible. To change scale, edit
`N_MEMBERS` at the top of the script.

## Files

| File | Rows | Description |
|------|-----:|-------------|
| `his_population.csv`         |  1,000 | Member master (demographics, household, observation window) |
| `his_medical_institution.csv`|  ~3,000 | Member ↔ institution links with prefecture codes |
| `his_claims.csv`             | ~39,000 | One row per claim (outpatient/inpatient/DPC) |
| `his_claims_disease.csv`     | ~93,000 | Diagnoses per claim (ICD-10) |
| `his_claims_drugs.csv`       | ~49,000 | Prescriptions per claim (ATC) |
| `his_claims_medacts.csv`     | ~34,000 | Medical acts per claim (surgery/lab/imaging) |
| `his_health_checkup.csv`     |  ~2,000 | Annual checkup measurements |

## Schema (matches `ml_framework_appendix/appendix_E_data_preparation.sql`)

### his_population
`member_id, dataset_id, sex_code, birth_yyyymm, relation_code, first_obs_date, last_obs_date, household_id`
- `sex_code`: 1=M, 2=F
- `relation_code`: 1=head, 2=spouse, 3=child
- Observation period: 2017-01-01 ~ 2023-12-31

### his_medical_institution
`institution_id, member_id, visit_date, prefecture_code`
- `prefecture_code`: 01~47 (Tokyo=13 weighted heaviest)

### his_claims
`claim_id, member_id, dataset_id, claim_date, claim_type, care_days, total_points, institution_id`
- `claim_type`: 11=outpatient (~85%), 12=inpatient (~8%), 13=DPC inpatient (~7%)

### his_claims_disease
`claim_id, member_id, dataset_id, icd10_code, onset_date, suspect_flag`
- `suspect_flag`: 0=confirmed (~90%), 1=suspect (~10%)
- ICD-10 codes drawn from `appendix_A_charlson_icd10.csv` + common general codes

### his_claims_drugs
`claim_id, member_id, atc_code, prescription_date, days_supply`
- `atc_code` values from `appendix_B_atc_categories.csv` (30 categories)
- `days_supply`: 14 / 30 / 60 / 90

### his_claims_medacts
`claim_id, member_id, act_code, act_category, act_date`
- `act_category`: 50=surgery (~2%), 60=lab (~70%), 70=imaging (~28%)

### his_health_checkup
`member_id, checkup_date, height_cm, weight_kg, sbp_mmhg, dbp_mmhg, ldl_mgdl, hdl_mgdl, tg_mgdl, hba1c_ngsp, fbg_mgdl, smoke_status`
- `smoke_status`: 0=never, 1=current, 2=former

## Generation rules (internal consistency)

The generator works in two phases so downstream features are coherent:

1. **Latent patient profile** — for each member, sample comorbidity flags
   (DM, HTN, HLD, CHF, CKD, Cancer, COPD, CVA, MI, Dementia, Rheumatic, Liver,
   Metastasis, AIDS), smoking status, and BMI category. Prevalence is
   age-modulated (older members carry more conditions).
2. **Profile-driven rows** — the patient's flags drive
   - which ICD-10 codes appear in claims (DM → E10/E11, HTN → I10, Cancer → C18/C34/...)
   - which ATC codes get prescribed (DM → A10A/A10B, HTN → C09/C08/C07/C03, HLD → C10AA, Cancer → L01)
   - which checkup values are abnormal (DM → HbA1c 6.3~12, HTN → SBP 130~200, HLD → LDL 130~250)

Target prevalence (latent flags, ~observed):

| Condition | Target | Observed |
|-----------|-------:|---------:|
| DM        | 0.10 | 0.099 |
| HTN       | 0.25 | 0.268 |
| HLD       | 0.22 | 0.205 |
| CHF       | 0.03 | 0.026 |
| CKD       | 0.04 | 0.033 |
| Cancer    | 0.05 | 0.045 |
| COPD      | 0.06 | 0.060 |
| MI        | 0.02 | 0.019 |
| CVA       | 0.03 | 0.033 |
| Dementia  | 0.02 | 0.023 |

## How to use in the app

1. Open the ML Auto Flow app.
2. Add a `LoadData` module and point it at `data/jmdc_synthetic/his_population.csv`
   (or any of the other CSVs).
3. Chain JMDC modules (J1~J7), e.g. `JMDCCohortBuilder → JMDCSurvivalCompare`.
4. For the cohort SQL examples in `appendix_E_data_preparation.sql`, the same
   logic can be applied in pandas against these CSVs.

## Disclaimer

Synthetic data only. Distributions are loosely calibrated to public references
(Japan adult DM prevalence ~10%, HTN ~25%) but are **not** drawn from the real
JMDC database. Do **not** publish modeling results trained on this data as
representative of JMDC findings.
